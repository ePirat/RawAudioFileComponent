ReadMe for RawAudioFileComponent
--------------------------------

The RawAudioFileComponent project provides a template for creating a custom audio file component. The sample provides an example of how to implement the necessary methods to be a valid audio file system extension.

Sample Requirements:
-------------------
This sample project requires:

	Mac OS X v10.9 or later
	Xcode 5.0 or later

Known Issue:
-----------

There is a known analyzer issue where it detects a potential memory leak in
this project.
